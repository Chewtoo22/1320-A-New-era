# Wireframe Documentation

## Race HUD Screen

### ASCII Art Layout
```
+---------------------+
|   TIMER: 00:01:23   |
|---------------------|
|   POSITION: 1st     |
|                     |
|   SPEED: 120 km/h   |
|                     |
| [   MAP   ] [ MINI ]|
+---------------------+
```

### Component Descriptions:
- **Timer**: Displays the current race time.
- **Position**: Current position in the race.
- **Speed**: Current speed of the vehicle.
- **Map**: Interactive map with a 2D view of the track.
- **Mini-map**: A smaller view of the main map.

### Dimensions:
- Mobile: Width - 360px, Height - 640px
- Desktop: Width - 1440px, Height - 900px


## Garage/Showroom Screen

### ASCII Art Layout
```
+--------------------+
|   GARAGE SHOWROOM   |
|--------------------|
|  [ CAR 1 ] [ CAR 2 ]|
|  [ .: CAR .: ]     |
|                    |
|  [ SELECT CAR ]    |
+--------------------+
```

### Component Descriptions:
- **Garage Showroom**: Overview screen for selecting cars.
- **CAR 1 / CAR 2**: Thumbnails of the cars.
- **SELECT CAR**: Button for selecting chosen car.

### Dimensions:
- Mobile: Width - 360px, Height - 640px
- Desktop: Width - 1440px, Height - 900px
