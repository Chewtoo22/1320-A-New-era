# Wireframes: Garage and Showroom

## Garage Screen Layout
- **Header**  
  - Logo at the top left  
  - Navigation menu on the top right  

- **Main Section**  
  - Two main areas:  
    - **Vehicles Display area:**  
      - Grid layout with vehicle cards  
      - Each card includes:  
        - Vehicle image  
        - Vehicle name  
        - Price
        - Quick action buttons (view details, reserve, etc.)  
    - **Filter Options:**  
      - Dropdowns for selecting categories (e.g., make, model, price range)  
      - Search bar for quick access  

- **Footer**  
  - Contact information  
  - Social media links  

## Showroom Screen Layout
- **Header**  
  - Logo at the top left  
  - Navigation menu on the top right  

- **Main Section**  
  - **Showcase Area:**  
    - Large image or video of showroom  
    - Brief description or slogan

  - **Featured Vehicles Section:**  
    - Horizontal scrollable set of vehicle cards similar to the garage

- **Footer**  
  - Contact information  
  - Social media links  

## Notes
- Focus on a clean, user-friendly interface that enhances vehicle visibility.
- Utilize responsive design principles to ensure optimal viewing on all devices.
